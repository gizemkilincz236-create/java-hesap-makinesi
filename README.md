# Java ile Basit Hesap Makinesi

Bu proje, kullanıcının konsol üzerinden girdiği iki sayı ile seçilen matematiksel işlemi (Toplama, Çıkarma, Çarpma, Bölme) gerçekleştiren basit bir Java konsol uygulamasıdır.

## 🚀 Özellikler

* **Dört İşlem Desteği:** Toplama, çıkarma, çarpma ve bölme işlemleri.
* **Hata Yönetimi:** Bölme işleminde sıfıra bölünme hatası (`ArithmeticException`) kontrolü.
* **Hassas Hesaplama:** Bölme sonuçlarının ondalıklı (`double`) olarak gösterilmesi.
* **Kullanıcı Dostu:** `Scanner` sınıfı ile dinamik kullanıcı girişi ve `switch-case` yapısı ile menü yönetimi.

## 🛠️ Kullanılan Teknolojiler

* **Dil:** Java (JDK 8 veya üzeri)
* **Kütüphaneler:** `java.util.Scanner`

## 💻 Nasıl Çalıştırılır?

1. Projeyi bilgisayarınıza indirin veya klonlayın.
2. `HesapMakinesi.java` dosyasını bir terminal veya IDE (VS Code, IntelliJ, Eclipse vb.) üzerinde açın.
3. Kodu derleyin ve çalıştırın:
   ```bash
   javac HesapMakinesi.java
   java HesapMakinesi
